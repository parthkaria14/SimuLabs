Binary Search

Overview

Binary search is an efficient algorithm used to search for a specific element in a sorted array or list. It follows the divide-and-conquer approach by repeatedly dividing the search interval in half until the target element is found or determined to be absent.

Procedure

  1. Consider a sorted array or list of elements.
  2. Set the initial values of low and high indices to the first and last elements, respectively.
  3. While the low index is less than or equal to the high index:
    1. Calculate the middle index as the average of low and high indices.
    2. If the middle element is equal to the target, the search is successful. Return the middle index or any relevant information.
    3. If the middle element is greater than the target, set the high index to the middle index minus 1.
    4. If the middle element is less than the target, set the low index to the middle index plus 1.
  4. If the target element is not found, the search is unsuccessful. Return a suitable message indicating that the element is not present in the array or list.

Example

Here's an example of using binary search:

        int[] arr = {10, 20, 30, 40, 50, 60, 70};
        int target = 40;
        
        int result = binarySearch(arr, target);
        if (result != -1) {
            System.out.println("Element found at index: " + result);
        } else {
            System.out.println("Element not found.");
        }
