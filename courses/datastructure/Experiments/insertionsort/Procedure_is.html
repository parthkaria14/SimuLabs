Insertion Sort

Overview

Insertion sort is a simple comparison-based sorting algorithm that builds the final sorted array one element at a time. It works by repeatedly selecting an element from the unsorted portion of the array and inserting it into its correct position within the sorted portion.

Procedure

  1. Consider an array of elements.
  2. Start with the second element and iterate through the array:
    1. Compare the current element with the elements in the sorted portion of the array.
    2. If the current element is smaller, shift the larger elements one position to the right.
    3. Insert the current element at the appropriate position in the sorted portion.
  3. Repeat step 2 until the entire array is sorted.

Example

Here's an example of using insertion sort:

        int[] arr = {64, 25, 12, 22, 11};
        insertionSort(arr);
        
        System.out.println("Sorted array:");
        for (int num : arr) {
            System.out.print(num + " ");
        }
