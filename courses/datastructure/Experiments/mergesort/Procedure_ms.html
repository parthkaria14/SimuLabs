Merge Sort

Overview

Merge sort is a divide-and-conquer sorting algorithm that recursively divides the input array into smaller subarrays, sorts them, and then merges them back together to obtain the sorted array. It works by repeatedly dividing the array in half until individual elements are reached, and then merges the sorted subarrays back in a sorted order.

Procedure

  1. Consider an array of elements.
  2. Divide the array into two halves by finding the middle index.
  3. Recursively apply merge sort to each half:
    1. Divide the current half into two halves.
    2. Recursively apply merge sort to each of the divided halves.
  4. Merge the sorted halves back together:
    1. Create an auxiliary array to store the merged result.
    2. Compare the elements from both halves and place them in sorted order in the auxiliary array.
    3. If one half is exhausted, copy the remaining elements from the other half to the auxiliary array.
    4. Copy the merged elements from the auxiliary array back to the original array.
  5. Repeat steps 2-4 until the entire array is sorted.

Example

Here's an example of using merge sort:

                    int[] arr = {64, 34, 25, 12, 22, 11, 90};
                    mergeSort(arr, 0, arr.length - 1);
                    
                    System.out.println("Sorted array:");
                    for (int num : arr) {
                        System.out.print(num + " ");
                    }
